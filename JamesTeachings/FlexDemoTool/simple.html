Flex Layout Model

  • Overview of Flex

    The contents of a Flex container:
    • can be layout in any flow direction
    • can have their display order reversed or reaaranged at the style layer
    • can be laid out linearly along a main axis or wrapped into multiple lines along a cross axis
    • can flex their sizes to respond to the available size
    • can be aligned with respect to their container or each other on the cross axis
    • can be dynamically collapsed or uncollapsed along the main axis while preserving the container's cross size
